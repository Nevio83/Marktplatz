Adresse ändern

Wie kann ich meine Adresse ändern?

Sie können Ihre Lieferadresse auf verschiedene Weise ändern:

Vor der Bestellung:

  • Gehen Sie zu Ihrem Kundenkonto
  • Wählen Sie "Adressen verwalten"
  • Fügen Sie eine neue Adresse hinzu oder bearbeiten Sie eine bestehende
  • Setzen Sie die neue Adresse als Standard-Lieferadresse

Während der Bestellung:

  • Im Warenkorb können Sie die Lieferadresse ändern
  • Klicken Sie auf "Adresse ändern"
  • Wählen Sie eine gespeicherte Adresse oder geben Sie eine neue ein

Nach der Bestellung:

  • Kontaktieren Sie unseren Kundenservice
  • Geben Sie Ihre Bestellnummer an
  • Wir versuchen, die Änderung vor dem Versand vorzunehmen

Hinweis: Adressänderungen sind nur möglich, solange die Bestellung noch nicht versendet wurde.
